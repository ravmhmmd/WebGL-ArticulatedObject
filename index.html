Texture

Untuk memilih texture object yang diinginkan, gunakan dropdown list pada "Current Texture". Terdapat 3 pilihan texture yaitu Image, Environtment, dan Bump

Texture : Image

Pada saat menggunakan texture Image, untuk memilih Image yang akan digunakan pada texture gunakan dropdown list pada "Current Image". Terdapat 3 pilihan Image yaitu Wall, Bump Wall, dan Rock

Texture : Bump

Pada saat menggunakan texture Bump, untuk memilih Bump Surce yang akan digunakan pada texture gunakan dropdown list pada "Current Bump Source". Terdapat 3 pilihan Image yaitu Bump Normal, Bump, dan Bump Skin

Rotasi

Untuk melakukan rotasi, gunakan slider "Rotate Along Axis" dan terdapat 3 rotasi terhadap sumbu x, y, dan z

Shader Mode

Untuk mengaktifkan shader mode tekan checkbox bertulisan "active" pada "Shader and Texture Mode", jika centang maka shader mode aktif

Animation

Untuk mengaktifkan animation tekan checkbox bertulisan "active" pada "Animation", jika centang maka animation mode aktif

Camera Handler

Untuk mengatur sudut penglihatan, dapat mengatur slider dengan tulisan "Angle"

Sedangkan untuk mengatur kamera untuk memperbesar atau memperkecil view, tekan tombol "Zoom In" untuk memperbesar / mendekat dan "Zoom Out" untuk memperkecil / menjauh

Object Part

Untuk melihat gerakan bagian setiap part dari object, gunakan slider sesuai part yang ingin digerakkan

Utilities

Untuk melakukan penyimpanan model bentuk, tekan tombol "Save" dan file berekstensi json akan menyimpan model tersebut

Untuk dapat memuat model dari file, dapat gunakan tombol input file terlebih dahulu dan pilih file model berekstensi json kemudian tekan tombol load untuk menggambar model dari file tersebut
